Crear proyecto

Para crear un proyecto en DevSoft Projects, sigue estos pasos:

Tutorial rápido

  1. Ve al menú Proyectos > Nuevo (si está disponible en tu versión del formulario).
  2. Escribe el nombre del proyecto en el campo Nombre del proyecto.
    Ejemplo: App DAM 2026
  3. Pulsa el botón Guardar para confirmar.

Acceso contextual (F1)

Esta ayuda es contextual:

  • Coloca el cursor dentro del campo Nombre del proyecto.
  • Pulsa F1 para abrir directamente esta página.
Nota: Si al pulsar Guardar el campo está vacío, la aplicación puede mostrar un aviso. Asegúrate de introducir un nombre válido.
