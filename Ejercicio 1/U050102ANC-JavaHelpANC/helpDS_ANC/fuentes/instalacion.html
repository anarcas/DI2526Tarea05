Estructura de instalación

Esta sección explica cómo se organiza la aplicación para su distribución e instalación (por ejemplo, usando Launch4j para generar el .exe y NSIS para crear el instalador).

¿Qué se instala?

Una instalación típica de DevSoft Projects incluye:

  • El ejecutable DevSoftProjects.exe (generado desde el .jar).
  • La carpeta lib/ con las librerías necesarias.
  • La carpeta helpDS/ con la ayuda JavaHelp (HTML, toc, índice y búsqueda).

Estructura recomendada

Ejemplo de estructura dentro de la carpeta de instalación:

DevSoftProjects/
  DevSoftProjects.exe
  lib/
    (librerías necesarias)
  helpDS/
    help_set.hs
    map.jhm
    toc.xml
    indice.xml
    fuentes/
      general.html
      crear_proyecto.html
      instalacion.html
    JavaHelpSearch/
      (archivos generados por el indexador)
Importante: Para que la pestaña Buscar funcione en JavaHelp, es necesario generar los archivos de JavaHelpSearch/ mediante la herramienta de indexación.
